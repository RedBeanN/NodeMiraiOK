# NodeMiraiOK

快速部署基于 `NodeJS` 的 [mirai](https://github.com/mamoe/mirai) 机器人项目

## Usage

### 安装 nmok

``` sh
npm i -g node-mirai-ok
```

### 创建项目

``` sh
nmok create my-qq-bot
cd my-qq-bot
nmok run
```

## 其他

### 初始化 `node-mirai-sdk` 或 `mirai-ts` 项目模板

``` sh
nmok init <-t>
```

### 添加 `mirai-console` 插件

``` sh
nmok add mirai-api-http
```
